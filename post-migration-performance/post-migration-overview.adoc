= Erfahren Sie mehr über ONTAP FLI nach der Migration
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Dieser Abschnitt beschreibt die Vorgehensweise nach Abschluss der Foreign LUN Import (FLI) Migration. Es gibt allgemeine Bereinigungsschritte, Schritte zur Leistungsvalidierung sowie Schritte für jeden Hosttyp (AIX, ESXi, HP, Linux, Solaris und Windows).

Die Schritte nach der Migration sind wichtig, um sicherzustellen, dass die Umgebung in einem sauberen und unterstützbaren Zustand hinterlassen wird und um zu überprüfen, ob die Migration sowohl aus Sicht der Datenintegrität als auch der Performance erfolgreich war.
